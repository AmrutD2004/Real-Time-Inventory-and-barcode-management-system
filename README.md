# Real-Time Inventory & Barcode Management System

A full-stack enterprise inventory management platform designed to manage products, warehouses, stock movements, barcode scanning, real-time inventory synchronization, notifications, and analytics.

## Features

### Authentication & Role-Based Access Control (RBAC)

* JWT Authentication
* Secure Password Hashing
* Refresh Token Support
* Role-Based Access Control
* Protected Routes & APIs

### Product Management

* Product CRUD Operations
* Category Management
* Product Image Uploads
* SKU Generation
* Product Search & Filtering

### Warehouse & Inventory Management

* Multiple Warehouse Support
* Stock Tracking
* Inventory Transfers
* Stock Movement History
* Low Stock Monitoring

### Barcode Scanning

* Barcode & QR Code Generation
* Camera-Based Barcode Scanning
* Instant Product Lookup
* Mobile-Friendly Scanner Interface

### Real-Time Updates

* Socket.IO Integration
* Live Inventory Synchronization
* Instant Stock Updates
* Real-Time Dashboard Refresh

### Notifications

* Low Stock Alerts
* Inventory Activity Notifications
* System Event Notifications
* Real-Time Notification Delivery

### Reporting & Analytics

* Inventory Dashboard
* Stock Movement Analytics
* Warehouse Reports
* Product Reports
* Excel Import & Export
* Interactive Charts & Graphs
